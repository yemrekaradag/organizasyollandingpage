# Organizasyol - Landing Page 🚀

Organizasyol uygulaması için tasarlanmış, modern, yüksek performanslı ve tamamen duyarlı (responsive) tanıtım sayfası (Landing Page). Proje, kullanıcıların ilgisini çekecek dinamik animasyonlar ve şık bir karanlık (dark mode) tasarımıyla geliştirilmiştir.

## 🌟 Özellikler

- **Modern & Premium Tasarım:** Etkileyici mor tema ve Glassmorphism (buzlu cam) efektleriyle "wow" dedirten kullanıcı arayüzü.
- **Pürüzsüz Animasyonlar:** Framer Motion ile desteklenen sayfa geçişleri, hover efektleri ve kaydırma animasyonları.
- **%100 Responsive:** Tüm mobil, tablet ve masaüstü cihazlarla kusursuz uyum.
- **SEO & Performans:** Meta etiketleri, Open Graph sosyal medya optimizasyonu, Robots.txt ve görseller için gecikmeli yükleme (Lazy Loading) altyapısı ile arama motoru dostu.
- **Hızlı Yükleme:** Vite kullanılarak yıldırım hızında derleme ve çalışma performansı.

## 💻 Kullanılan Teknolojiler (Tech Stack)

- **[React](https://reactjs.org/)** (v18+)
- **[Vite](https://vitejs.dev/)** (Hızlı derleyici ve geliştirme ortamı)
- **[Framer Motion](https://www.framer.com/motion/)** (Animasyonlar)
- **[Lucide React](https://lucide.dev/)** (Modern İkon Paketi)
- **Vanilla CSS** (Özelleştirilmiş esnek tasarım mimarisi)

## 🛠️ Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyin:

1. **Projeyi Klonlayın (veya indirin):**
   ```bash
   git clone https://github.com/yemrekaradag/organizasyollandingpage.git
   cd organizasyol-landing-page
   ```

2. **Gerekli Paketleri Yükleyin:**
   ```bash
   npm install
   ```

3. **Geliştirme Sunucusunu Başlatın:**
   ```bash
   npm run dev
   ```

4. **Tarayıcıda Görüntüleyin:**
   Terminalde beliren adresi (genellikle `http://localhost:5173`) tarayıcınızda açın.

## 📦 Canlıya Alma (Build)

Projeyi yayınlamaya hazır hale getirmek için:
```bash
npm run build
```
Bu komut, `dist/` klasörü içerisinde optimize edilmiş, yayına hazır (production-ready) dosyaları oluşturur.

## 🖼️ Projeyi Özelleştirme
- **Resimler:** `src/components` altındaki `.jsx` dosyalarındaki geçici Unsplash resim linklerini, kendi uygulamanızın gerçek ekran görüntüleriyle (mockup) değiştirebilirsiniz.
- **Linkler:** Footer ve Header içerisindeki yönlendirme (Store ve sosyal medya) bağlantılarını güncelleyebilirsiniz.
